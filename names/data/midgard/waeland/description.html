Country description

Like many other Midgard countries, Waeland is inspired by an existing country in human history. In this case, it's the vikings.

Historical

Regional

There are some regional gods. Maybe some parts of the names of these gods are put into the names of people. The gods are:
  • Asvargr War
  • Fjörgynn Fertility
  • Vidar Creator of the world
  • Wyrd Wisdom
There are more. But I don't think the others will become part of the name/nickname.

Usage

Building names

Names are build like western european names, sometimes you do not have a family name or the family name is subsituted by a "Beiname" (additional name). The "Beinamen" are given to the Waeländer by friends or family. It bases on body attributes ("Einauge"="One eye") or events (good and bad) happening.
Family can be the simple names of the clan or the first name of mother or father with the attached -son (for son) or -dottir/-daughdir for daughter.
